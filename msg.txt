fix(ui): correct flex and grid intrinsic height overflow in gallery

Resolved a layout bug where the intrinsic height of the thumbnail images
forced the parent flex and grid containers to expand beyond their set
height of 500px. This overflow caused the gallery row to overlap and
completely hide the property title, badges, and address below it.
Added min-h-0 to the grid column, flex container, and flex items to
allow CSS flexbox and grid engines to correctly shrink the items within
the bounded 500px container.
